# Qubena Macro
**学習サイトQubenaを半自動的に回答させる拡張機能**

## 使い方
- 1.Releaseフォルダにあるextentions.zipをインストールし、解凍します。
- 2.使用したいChromeプロファイルの拡張機能タブを開きます。(chrome://extensions/)
- 3.デベロッパーモードを有効にし、回答したフォルダをそのままドラッグアンドドロップします。
- 4.エラーが出ず、正常にインポートできたことを確認してください。
- 5.[Google AI Studio](https://aistudio.google.com/api-keys)からAPIキーを取得します。
- 6.拡張機能を開き、先ほど取得したAPIキーを入力し、Qubenaのタブを開きます。
- 7.拡張機能のプロンプト入力欄に指示を書き、そのまま実行します。
